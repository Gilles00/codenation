# FINAL PROJECT (Loan Management System)
## Objetivo / Objective
* O principal objetivo deste projeto é criar uma API para gerenciar o sistema de controle de pagamentos de empréstimos de uma fintech.

* The main objective of this project is to create an API to manage the loan payments control system from a fintech.

## Propósito / Purpose
* O objetivo deste desafio é testar sua capacidade de implementar uma solução devido a um problema abstrato. Você pode encontrar um problema na tarefa que você precisa para explicar o problema e propor uma solução para solucioná-lo.

* The purpose of this challenge is to test your ability to implement a solution given an abstract problem. You may find a problem in the asked task that you need to explain the problem and propose a solution to fix it.

## Problema / Problem
* Uma fintech precisa acompanhar a quantidade de dinheiro emprestado e os pagamentos perdidos / feitos. Ele também precisa de um lugar para recuperar o volume de dívida pendente em algum momento.

* A fintech needs to keep track of the amount of money loaned and the missed/made payments. It also needs a place to retrieve the volume of outstanding debt at some point in time.

## Limitações / Limitations
* Os empréstimos são pagos em parcelas mensais.

* Loans are paid back in monthly installments.
